# AVON BOT

## Overview
AVON BOT is a full Discord bot system that includes:
- Credits system
- Captcha-secured transfers
- Tax system (5%)
- Premium memberships
- AVON GPT (AI chat system)

## Core Features

### Credits System
- Transfer command:
  C @user amount
- Captcha confirmation
- 5% tax on transfers
- Transfer channel set by slash command

### Premium System
- Plus
- Premium
- Max
- Premium users are exempt from tax
- Premium affects limits and GPT usage

### AVON GPT
- AI chat in specific channels
- Word limits based on plan
- Separate channels for text & images
- GPT usage tracking per user

## Permissions
- Owner: full access
- Admin: management access
- Member: normal usage
